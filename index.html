SOUTHERN LUZON STATE UNIVERSITY CAMPUS MAP

Build with HTML, CSS, Bootstrap, PHP, JavaScript
Visual Studio and XAMPP application
The SLSU Campus Map is a webpage with an interactive map that helps users to view and locate any SLSU Satellite Campus, as well as to see some of the information's per campuses like: list of total enrolled student, list of available courses, and as well as the exact coordinate of the campus. There are 2 modules on this webpage namely admin and user.
The admin can add, edit and delete the campus information such as its coordinates, address, its total number of staff, the total number of students enrolled, available courses, and as well as some photos of the campus.
As for the users, the user interacts with the map on the webpage, once the user hovered over the interactive map markers it will show the campus brief information as well as the campus logo. The user can also click the marker in order to view the full details of the campus.

ONLINE COLLEGE ADMISSION MANAGEMENT SYSTEM

Build with HTML, CSS, Bootstrap, PHP, JavaScript
Visual Studio and XAMPP application
The College Admission Management Systemaims to make the admission process much easier and helps in maintaining the database in an efficient way. There are 2 modules in this system namely admin and user (students).
Admin can add, edit and delete the college details, courses, staff, and faculty personnel, it can also show the number of staff and courses available in the college. The system can show the number of students that registered on the website, it shows the student's name, students type, and also the course that they want to apply. The system can view, edit, delete or print out the student’s information.
The users (students) can log in and fill up the blank boxes with their personal information, and the school they attended last. The user (students) can also attach some files for their birth certificate, last school grades, and other documents that are required when applying for college.

MOBILE -BASED TABULATION SYSTEM FOR SCHOLASTIC COMPETITION

Build with Java, PHP and MySQL
Android Studio, Netbeans and XAMPP application
The Mobile-Based Tabulation System for Scholastic Competition aims to help the tabulation team in computing and generating the results of the event in a short amount of time also in setting up and configuring the system during pageantry and other scholastic event. It also aims to provide ease of access through a user-friendly interface.
This Tabulation System is one of the projects of Southern Luzon State University-Lucena Campus
